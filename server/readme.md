# Express + TypeScript Server

Jednoduchý backend postavený na Express s podporou TypeScriptu.

## 💻 Spustenie lokálne

1. Nainštaluj závislosti:
  ```bash
  npm install

1. Spusti server v development režime:
  ```bash
  npm run dev

Server beží na: http://localhost:3000

📦 Použité technológie

Express
TypeScript
ts-node-dev
